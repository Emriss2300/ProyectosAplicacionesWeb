Bienvenido a la página de Noticias

En esta página web, podrás ver y administrar noticias. Al acceder a la página "index.php", encontrarás una lista de noticias disponibles en orden descendente según su ID. Cada noticia muestra su titular, fecha, descripción y una imagen asociada.

Visualizar Noticias

Al abrir la página "index.php", podrás ver todas las noticias existentes. Cada noticia está presentada de forma ordenada y organizada para una fácil lectura. Si no hay noticias disponibles, el sistema te informará que no se encontraron noticias y te dara la opcion de agregar noticias.

Administrar Noticias

Si deseas administrar las noticias, puedes acceder a la página "admin.php" directamente. En esta página, podrás agregar, editar y eliminar noticias, cabe destacar que esta es una pagina a la que puedes acceder desde la pagina principal index.php cuando quieras realizar alguna de las acciones indicadas anteriormente.

Para agregar una nueva noticia, simplemente haz clic en "Crear Noticia" y completa los campos requeridos: fecha, titular, descripción y sube una imagen. Una vez creada, la noticia aparecerá en la página de inicio.

Si deseas editar una noticia existente, haz clic en "Editar" junto a la noticia que deseas modificar. Se te redirigirá a una página donde podrás editar los campos de la noticia. Si deseas cambiar la imagen, también puedes hacerlo.

Para eliminar una noticia, haz clic en "Eliminar" junto a la noticia que desees eliminar. Una vez confirmado, la noticia será eliminada de la base de datos y ya no aparecerá en la página de inicio.

Importante

Para poder utilizar esta página eficientemente, necesitarás tener un entorno adecuado para ejecutar PHP y MySQL. Asegúrate de tener instalado un servidor web (por ejemplo, Apache) y un servidor de bases de datos MySQL. También asegúrate de tener PHP habilitado en tu servidor web.

Además, dentro de la carpeta contenedora de este proyecto, encontrarás un archivo SQL llamado "validar.sql". Este archivo contiene el script SQL que se utilizó para crear la base de datos y la tabla de noticias necesarias para que el sistema web funcione correctamente. Asegúrate de importar este archivo SQL en tu servidor de bases de datos para tener una estructura adecuada para el funcionamiento del sistema.

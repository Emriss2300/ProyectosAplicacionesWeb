¡Bienvenido a la página de Noticias!
Reestructuramos la plataforma web siguiendo el patrón Modelo-Vista-Controlador (MVC), lo que hará que tu experiencia sea más eficiente y amigable.

Aquí podrás disfrutar de la visualización y administración de noticias de manera sencilla. Al acceder a "index.php", encontrarás una lista de las noticias disponibles, organizadas en orden descendente según su ID. Cada noticia muestra su titular, fecha, descripción y una imagen asociada, brindándote una experiencia de lectura agradable y práctica.

Visualizar Noticias

Al abrir la página "index.php", te encontrarás con todas las noticias existentes, presentadas de forma ordenada y clara para su lectura. En caso de no haber noticias disponibles, el sistema te informará amablemente y te brindará la opción de agregar noticias.

Administrar Noticias

Para administrar las noticias, simplemente accede directamente a "admin.php" y como usuario al boton administrar noticias. En esta sección, podrás realizar tres acciones esenciales: agregar, editar y eliminar noticias. Es importante mencionar que puedes acceder a esta página desde la principal "index.php" cuando desees realizar alguna de estas acciones.

Si deseas agregar una nueva noticia, haz clic en "Crear Noticia" y completa los campos requeridos: fecha, titular, descripción y sube una imagen. Una vez creada, la noticia aparecerá en la página de inicio.

En caso de que necesites editar una noticia existente, selecciona "Editar" junto a la noticia que desees modificar. Se te redirigirá a una página donde podrás editar los campos de la noticia, y si deseas cambiar la imagen, también podrás hacerlo de forma sencilla.

Por otro lado, si deseas eliminar una noticia, solo tienes que hacer clic en "Eliminar" junto a la noticia que desees eliminar. Una vez confirmado, la noticia será eliminada de la base de datos y dejará de aparecer en la página de inicio.

Información Importante

Para hacer uso eficiente de esta página, necesitarás contar con un entorno adecuado para ejecutar PHP y MySQL. Asegúrate de tener instalado un servidor web, como Apache, y un servidor de bases de datos MySQL. Además, verifica que PHP esté habilitado en tu servidor web.

Para acceder a la base de datos "validar", utiliza las siguientes credenciales:

Host: localhost
Usuario: Francisco1234
Contraseña: 1234
Nombre de la base de datos: validar

Además, en la carpeta del proyecto, encontrarás un archivo SQL llamado "validar.sql". Este archivo contiene el script SQL que crea la base de datos y la tabla de noticias necesarias para el correcto funcionamiento del sistema. Importa este archivo SQL en tu servidor de bases de datos para contar con la estructura adecuada.

Cabe mencionar que el archivo "index.php" se ha dejado fuera de las carpetas para servir como guía de uso del sistema web.

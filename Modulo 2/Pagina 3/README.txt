¡Hola! Mi nombre es Francisco Núñez y este es mi sistema web para generar la serie Fibonacci. Permíteme explicarte cómo funciona.

La página principal de mi sistema se llama "index.php". En esta página, los usuarios pueden ingresar un número entre 1 y 99 y luego mostraré la serie Fibonacci hasta ese número.

Al cargar la página, verás un encabezado con el título "Serie Fibonacci". He aplicado estilos a la página utilizando un archivo CSS llamado "estyles.css". Este archivo CSS se encuentra en la misma carpeta que el archivo "index.php".

A continuación, encontrarás un formulario que solicita al usuario ingresar un número. El formulario se envía utilizando el método POST para mantener la seguridad de los datos.

Cuando el usuario envía el formulario, mi sistema verifica si el número ingresado es válido. Si el número es válido, genero la serie Fibonacci hasta ese número y la muestro en pantalla. Si el número no es válido, muestro un mensaje de error para que el usuario ingrese un número válido entre 1 y 99.

He utilizado la función "fibonacci($n)" para generar la serie Fibonacci. Esta función recibe el número ingresado por el usuario y utiliza un bucle "for" para calcular cada número de la serie. Los números se almacenan en un array y luego se muestran en pantalla.

Para personalizar la apariencia de la página, he cambiado el fondo a un color rosa utilizando el estilo en el archivo "estyles.css".

Recuerda que las imágenes relacionadas con la página se encuentran en la carpeta contenedora junto con los archivos PHP y CSS.
